# Training Knowledge Vault — Gemini / Antigravity adapter

The canonical, tool-neutral instructions for this vault live in **`AGENTS.md`**. Read that file first.

- **Google Antigravity** loads `AGENTS.md` and `GEMINI.md` automatically at the repo root. It already has `AGENTS.md` — this file is only for Gemini/Antigravity-specific *overrides*, so it is intentionally short to avoid loading the same content twice.
- **Gemini CLI** loads `GEMINI.md` by default; `.gemini/settings.json` in this repo adds `AGENTS.md` to `context.fileName` so the canonical instructions load too.

Keep this file empty of general vault rules. Add a Gemini- or Antigravity-specific override here **only** when one tool genuinely needs different behaviour from the rest — and note why.

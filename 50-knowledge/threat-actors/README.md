# Threat Actors

## Purpose

This section holds structured profiles for threat actors encountered in investigations. Each profile follows `_template.md` and is graded against source reliability using NATO Admiralty Codes.

## How to Add a New Profile

1. Copy `_template.md` to `{actor-slug}.md`
2. Fill in all mandatory sections (Executive Summary, Identity, Motive, Victimology, MO)
3. Grade all sources in Section 9 using Admiralty Codes (A–F for source, 1–6 for information)
4. Set `status: draft` until the profile has been reviewed by a second analyst
5. Update this index

## Admiralty Code Quick Reference

| Source Grade | Meaning |
|---|---|
| A | Completely reliable |
| B | Usually reliable |
| C | Fairly reliable |
| D | Not usually reliable |
| E | Unreliable |
| F | Cannot be judged |

| Information Grade | Meaning |
|---|---|
| 1 | Confirmed by other sources |
| 2 | Probably true |
| 3 | Possibly true |
| 4 | Doubtful |
| 5 | Improbable |
| 6 | Cannot be judged |
